# Nord AI v5 — Autonomous Agent System

![Nord AI](logo.svg)

## Vad är Nord AI?

Nord AI är ett autonomt multi-agent-system med 8 specialiserade agenter, 54 moduler, kognivt minne, Theory of Mind, WASI-sandbox, parallel reasoning, och säkerhet enligt 8 internationella standarder. Helt provider-agnostisk.

## 8 Specialiserade Agenter
| Agent | Roll |
|---|---|
| Orchestrator | Ledare — routing, syntes, Confidence Router |
| Researcher | Forskning — informationssammanställning |
| Coder | Kod — skriva, felsöka, k=3 kandidater, auto-correct |
| Analyzer | Analys — Chain-of-Thought, Tree-of-Thought |
| Security | Säkerhet — OWASP, GDPR, AI Act |
| Writer | Innehåll — dokumentation, rapporter |
| DevOps | Infrastruktur — Docker, CI/CD |
| Reviewer | Kvalitet — kodgranskning, testförslag |

## 54 Moduler
Kärnsystem, säkerhet, kognition, produktion, orkestrering, funktioner, juridik, säkerhetstest, lärande, tillgång, DevOps

## Säkerhet (8 standarder)
OWASP, GDPR, EU AI Act, ISO 27001, SOC 2, NIST CSF, NIST AI RMF, CIS Controls v8

## Juridik
EU AI Act, GDPR, eIDAS 2.0, CRA, Data Act, Brottsbalken, Försäkringsavtalslagen, Socialförsäkringsbalken, Nordisk implementering (SE/NO/DK/FI/IS)

## Snabbstart
```bash
docker-compose up -d
# eller
cd backend && pip install -r requirements.txt && python main.py
```

## Provider-agnostisk
Inga LLM-varumärken. Konfigurera PROVIDER_A_KEY, PROVIDER_B_KEY, LOCAL_URL i .env
